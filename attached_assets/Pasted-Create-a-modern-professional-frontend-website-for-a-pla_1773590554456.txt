Create a modern, professional frontend website for a platform called "CarConnect".

CarConnect is a digital automobile service platform that connects customers and automobile service providers. The platform already has a mobile app where users can book services and make payments. This website will act as a marketing and service showcase website with an attractive landing page and smooth user experience.

Website Requirements:
- Frontend only (no backend needed)
- Modern UI/UX
- Fully responsive (mobile + tablet + desktop)
- Smooth animations and transitions
- Attractive car-related visuals
- Use carousel sliders to showcase services and products
- Theme color: Black background with Light Green accents
- All buttons, hover effects, icons, and highlights should use Light Green color
- Clean typography and modern layout
- Add subtle motion effects and scroll animations

Brand Name:
CarConnect

Tagline:
"Your One-Stop Automobile Service Platform"

Main Purpose of Website:
1. Showcase services provided in the CarConnect platform
2. Showcase product categories
3. Attract customers to use the app
4. Allow business owners to take dealership and join the platform
